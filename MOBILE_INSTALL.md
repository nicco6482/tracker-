# 📱 Instalar Tracker de Gastos en tu Móvil

Tu app es una **PWA (Progressive Web App)** - Funciona completamente offline con acceso a botón de home screen como nativa.

## 🚀 Instalación

### En Android:
1. Abre la app en Chrome/Edge en tu móvil
2. Toca el **menú ⋮ (tres puntos)** en la esquina superior derecha
3. Selecciona **"Instalar aplicación"** o **"Agregar a pantalla de inicio"**
4. ¡Listo! La app aparecerá como un ícono en tu pantalla de inicio

### En iPhone:
1. Abre Safari
2. Navega a tu sitio web donde está la app
3. Toca el **botón Compartir** (cuadro con flecha)
4. Selecciona **"Agregar a pantalla de inicio"**
5. Dale un nombre (ej: "Gastos")
6. ¡Listo! Aparecerá como app en tu pantalla

## 🔑 Funcionalidades

### ✅ Funciona SIN Internet:
- ✓ Ver todos tus gastos guardados
- ✓ Agregar nuevos gastos (se guardan localmente)
- ✓ Ver cartera y saldo disponible
- ✓ Ver gráficos y estadísticas
- ✓ Editar y eliminar gastos

### 🌐 Con Internet:
- ✓ Todo lo anterior +
- ✓ Usar el chatbot con IA (Groq)
- ✓ Registrar gastos por chat natural

## 💾 Almacenamiento

- **Todos los datos se guardan localmente** en tu teléfono
- Gastos, cartera, API Key - todo privado
- Nunca se envía información a servidores (excepto mensajes al API de Groq)
- Si borras la app, puedes reinstalarla y recuperar tus datos (están en el navegador)

## 🔔 Indicador de Estado

- **🟢 En línea**: Conexión disponible
- **🟠 Sin conexión**: Solo funciones offline disponibles

## ❓ FAQ

**P: ¿Dónde se guardan mis datos?**  
R: Todo está en tu móvil, localmente en el navegador. Nada se envía a servidores externos.

**P: ¿Qué pasa si desinstalo la app?**  
R: Los datos siguen en el navegador. Si reinstales, todo seguirá ahí.

**P: ¿Puedo acceder desde otro dispositivo?**  
R: No, los datos están solo en ese dispositivo. Si quieres sincronizar, habrá un botón de exportar próximamente.

**P: ¿Funciona sin WiFi?**  
R: Sí, pero el chatbot necesita internet. El resto funciona perfectamente sin conexión.
